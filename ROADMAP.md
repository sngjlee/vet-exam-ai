## Phase 1 — 핵심 학습 기능 강화

> 수험생이 매일 쓰는 기능. 바로 체감 가능.

### 1. 과목 선택 퀴즈
- 원하는 과목(약리학, 내과학 등)을 골라서 퀴즈 시작
- 복수 선택 가능 (예: "약리학 + 병리학 30문제")
- 현재 전 과목 랜덤만 가능 → 약점 집중 학습 불가

### 2. 모의고사 모드
- 실제 시험과 동일한 문제 수 / 과목 배분
- 시간 제한 타이머 (전체 시험 기준)
- 제출 후 과목별 점수 리포트
- 실전 감각 훈련에 필수

### 3. 시험일 D-day + 학습 플랜
- 수의사 국가고시 날짜 입력 → 남은 일수 표시
- 하루 권장 문제 수 자동 계산
- 대시보드 최상단에 고정 노출

---

## Phase 2 — 학습 유지 및 품질

> 꾸준히 쓰게 만드는 기능.

### 4. 학습 스트릭 강화
- 연속 학습 일수 배지 + 시각적 강조
- 오늘 목표 달성 여부 (N문제 완료)
- 현재 대시보드에 일부 있으나 존재감 약함

### 5. 문제 북마크
- 정답이어도 다시 보고 싶은 문제 저장
- 오답 노트와 별개의 "즐겨찾기" 리스트
- 북마크 모아서 복습하는 세션

### 6. 문제 신고 / 오류 제보
- 문제 오류, 해설 오류 신고 버튼
- 관리자 검토 후 수정 반영
- 문제 품질 유지에 필수

---

## Phase 3 — 확장 기능

> 사용자 기반이 생긴 후 고려.

### 7. 복습 알림
- 오늘 복습할 문제가 있을 때 이메일 알림
- 브라우저 Push 알림 (선택적)

### 8. 친구 / 학습 비교
- 친구 초대 링크
- 주간 문제 수 / 정답률 비교
- 학습 동기 부여용

### 9. AI 해설 강화
- 기존 해설 외에 AI가 추가 설명 생성
- "왜 틀렸는지" 개인화 피드백
