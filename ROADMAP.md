## Phase 0 — 문제은행 신뢰도 확보

> 수험생이 "이걸 풀면 실제로 도움이 된다"고 믿게 만드는 기반.  
> 현재 진행 중인 기출 파싱 → AI 재작성(Sonnet 4.6) → 검수 → Supabase 삽입 작업.

### 1. AI 재작성 결과물 검수
- 과목별 샘플 5~10문제씩 원문 대비 확인
- 정답 유지 여부, 선택지 중복/모순, 해설 정확도, 난이도 변형 여부 점검
- 이미지 포함 문제는 1차에서 `is_active=false`로 분리

### 2. 자동 검증 스크립트
- `choices.length === 5` 확인
- `answer`가 choices 중 하나와 정확히 일치하는지 확인
- 문제/선택지/정답/해설 빈 값 검사
- `id`, `subject`, `category`, `session`, `round`, `year` 누락 및 중복 검사
- 전체 JSON 검증 후 통과한 문제만 업로드 후보로 분류

### 3. Supabase 스키마 확장
- `questions` 테이블에 `session`, `round`, `year`, `community_notes`, `tags`, `is_active` 컬럼 추가
- `source="past_exam"` 기준으로 기출 기반 문제 구분
- `id` 기준 upsert 가능하도록 중복 정책 확정

### 4. 20과목 데이터 구조 반영
- 기존 5과목 중심 category를 수국시 20과목으로 확장
- 과목 선택, 통계, 약점 분석, 색상 팔레트가 20과목을 처리하도록 수정
- 교시별 필터(1교시/2교시/3교시/법규) 확장 고려

### 5. 소량 업로드 후 앱 플로우 테스트
- 예: `해부학` + `수의법규`처럼 성격이 다른 2개 과목 먼저 업로드
- 퀴즈, 오답노트, SRS 복습, 통계, 약점 집중 연습 정상 동작 확인
- 문제가 앱에서 학습 가능한 품질인지 실제 사용자 흐름으로 검증

## Phase 1 — 핵심 학습 기능 강화

> 수험생이 매일 쓰는 기능. 바로 체감 가능.

### 6. 과목 / 교시 / 문제 수 선택 퀴즈
- 원하는 과목(약리학, 내과학 등)을 골라서 퀴즈 시작
- 복수 선택 가능 (예: "약리학 + 병리학 30문제")
- 교시별 선택 가능 (예: "1교시 기초 과목만")
- 문제 수 선택 가능 (5 / 10 / 30문제 등)
- 현재 전 과목 랜덤만 가능 → 약점 집중 학습 불가

### 7. 시험일 D-day + 학습 플랜
- 수의사 국가고시 날짜 입력 → 남은 일수 표시
- 하루 권장 문제 수 자동 계산
- 남은 문제은행 기준 예상 회독 수 표시
- 대시보드 최상단에 고정 노출

### 8. 미니 모의고사 / 실전 모의고사 모드
- 1차: 과목별 또는 교시별 미니 모의고사
- 2차: 실제 시험과 동일한 문제 수 / 과목 배분
- 시간 제한 타이머
- 제출 후 과목별 점수 리포트
- 합격선 대비, 취약 과목 Top 5, 다시 풀 문제 추천
- 실전 감각 훈련에 필수

## Phase 2 — 학습 유지 및 품질

> 문제를 더 잘 기억하고, 문제은행 품질을 계속 높이는 기능.

### 9. 문제 신고 / 오류 제보
- 문제 오류, 해설 오류, 정답 오류 신고 버튼
- 관리자 검토 후 수정 반영
- 신고 많은 문제는 자동 비활성 후보로 표시
- 문제 품질 유지에 필수

### 10. 문제 북마크
- 정답이어도 다시 보고 싶은 문제 저장
- 오답 노트와 별개의 "즐겨찾기" 리스트
- 북마크 모아서 복습하는 세션

### 11. 학습 스트릭 강화
- 연속 학습 일수 배지 + 시각적 강조
- 오늘 목표 달성 여부 (N문제 완료)
- 현재 대시보드에 일부 있으나 존재감 약함

### 12. 수험생 팁 노출
- `community_notes`를 "수험생 팁" 섹션으로 노출
- 정정, 암기법, 보충 설명을 문제 하단에 표시
- 검수 전 댓글은 별도 표시 또는 관리자 승인 후 노출

## Phase 3 — 확장 기능

> 사용자 기반이 생긴 후 고려.

### 13. 복습 알림
- 오늘 복습할 문제가 있을 때 이메일 알림
- 브라우저 Push 알림 (선택적)

### 14. 친구 / 학습 비교
- 친구 초대 링크
- 주간 문제 수 / 정답률 비교
- 학습 동기 부여용

### 15. AI 해설 강화
- 기존 해설 외에 AI가 추가 설명 생성
- "왜 틀렸는지" 개인화 피드백
- 선지별 오답 이유, 관련 개념, 비슷한 함정 설명
- 검수된 기본 해설을 우선하고 AI 해설은 보조 설명으로 제공
