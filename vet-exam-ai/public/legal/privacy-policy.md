# 개인정보 처리 및 가입 정책

시행일: 2026년 4월 26일

본 문서는 수의미래연구소가 운영하는 KVLE의 개인정보 처리 원칙과 회원가입 정책을 정리한 공개용 운영 기준입니다. 수의미래연구소의 개인정보 보호책임자, 처리위탁, 국외 이전, 보유 기간, 파기 절차, 권리 행사 방법 등 서비스 운영에 필요한 세부 정보는 개인정보 처리방침과 서비스 화면에 별도로 고지합니다.

## 1. 기본 원칙

- 수의미래연구소는 KVLE 서비스 제공에 필요한 최소한의 개인정보만 수집합니다.
- 회원가입 초기에는 이메일과 비밀번호 기반 가입을 원칙으로 합니다.
- 이름, 휴대전화번호, 학교, 학번, 학생증, 재학증명서, 면허번호 등은 기능 제공 또는 법적 필요성이 명확해진 경우에만 선택적으로 수집합니다.
- 선택 항목을 입력하지 않아도 기본 학습 기능 이용에는 불이익이 없도록 설계합니다.
- 개인정보 수집 목적, 항목, 보유 기간, 동의 여부는 회원이 이해하기 쉬운 문장으로 가입 화면에 표시합니다.

## 2. 가입 시 필수 항목

초기 가입에 필요한 필수 항목은 다음과 같습니다.

- 이메일 주소
- 비밀번호
- 이용약관 동의
- 개인정보 처리방침 동의
- 만 14세 이상 여부 확인

서비스 이용 과정에서 다음 정보가 자동 생성될 수 있습니다.

- 회원 식별자
- 로그인 기록
- 문제 풀이 기록
- 오답노트, 복습 상태, 학습 통계
- 댓글, 암기법, 정정 제안, 신고 내역
- 접속 로그, 기기 및 브라우저 정보, IP 주소

자동 생성 정보는 개인정보 또는 개인정보에 준하는 운영 데이터로 관리하며, 수집 목적과 보유 기간을 문서화합니다.

## 3. 선택 수집 항목

다음 항목은 해당 기능을 제공할 때 별도 안내와 동의를 거쳐 선택적으로 수집합니다.

- 닉네임: 커뮤니티, 공개 프로필 또는 작성자 표시 기능 제공 시
- 시험 일정: D-day, 학습 계획, 알림 기능 제공 시
- 준비 시험: 수의사 국가시험, 동물보건사 시험 등 학습 트랙 구분 시
- 학교 또는 소속: 학교별 기능, 인증 배지, B2B 제휴 기능 제공 시
- 휴대전화번호: SMS 인증, 결제, 강한 본인확인 또는 계정 복구가 필요한 경우
- 증빙 자료: 학생 또는 전문가 인증이 필요한 기능을 제공하는 경우

선택 항목은 입력하지 않아도 필수 기능 이용이 가능해야 하며, 선택 항목 미입력으로 제한되는 기능은 가입 또는 입력 화면에서 명확히 안내합니다.

## 4. 동의 설계

- 이용약관 동의와 개인정보 처리방침 동의는 필수 동의로 분리합니다.
- 마케팅 수신, 맞춤형 분석, 연구·통계 목적 활용, 광고성 알림은 각각 필요한 경우에만 별도 선택 동의를 받습니다.
- 선택 동의는 기본값을 미동의로 두고, 필수 동의와 한 번에 묶어 동의시키지 않습니다.
- 가입 화면에는 처리 목적, 수집 항목, 보유 기간을 요약하고 전체 개인정보 처리방침으로 연결합니다.
- 동의 철회 방법은 동의 화면 또는 개인정보 처리방침에서 안내합니다.

## 5. 가입 자격

- 초기 서비스는 만 14세 이상 이용자를 대상으로 합니다.
- 만 14세 미만 가입은 지원하지 않습니다.
- 향후 만 14세 미만 가입을 허용하는 경우 법정대리인 동의, 아동 개인정보 안내, 별도 보호 절차를 마련합니다.
- 서비스는 수의사 국가시험 및 관련 학습자를 주요 대상으로 하나, 초기에는 학교나 자격을 필수 인증하지 않습니다.
- 인증 기반 기능을 도입하는 경우 인증 정보의 필요성, 수집 범위, 보관 기간, 삭제 절차를 별도로 안내합니다.

## 6. 자격 인증 정책

자격 인증은 다음 목적이 명확할 때만 도입합니다.

- 학교 또는 전문가 전용 기능 제공
- 유료 B2B 또는 기관 제휴 모델 운영
- 커뮤니티 신뢰도 확보 또는 악용 방지
- 법적 권리 또는 라이선스 조건상 접근 제한 필요

인증 방식은 개인정보 침해 가능성이 낮은 순서로 검토합니다.

1. 학교 이메일 인증
2. 일회성 수동 검수 후 인증 상태만 저장
3. 필요한 경우에 한해 최소 증빙 자료 제출

학생증, 재학증명서, 면허증 등 증빙 자료 원본은 검수 후 즉시 삭제하는 것을 원칙으로 하며, 인증 여부와 검수 시각 등 최소 정보만 보관합니다.

## 7. 학습 트랙과 권한 설계

장기 확장을 위해 가입 자격과 학습 트랙은 분리합니다.

- `user_role`: `learner`, `reviewer`, `admin`
- `exam_track`: `veterinarian`, `veterinary_technician`
- `verification_status`: `none`, `student_verified`, `professional_verified`, `manual_verified`

수의사 국가시험과 동물보건사 시험은 과목, 문항 구조, 학습 계획이 다르므로 `exam_track` 기준으로 분리 관리합니다.

## 8. 보유 기간과 탈퇴 처리

- 회원이 탈퇴하면 이메일, 계정 식별 정보 등 회원 식별 정보는 지체 없이 삭제하거나 법령상 보관이 필요한 경우 분리 보관합니다.
- 문제 풀이 기록, 오답노트, 복습 상태 등 학습 데이터는 회원 복구 필요성이 없고 법령상 보관 의무가 없는 경우 삭제하거나 익명 처리합니다.
- 결제, 환불, 분쟁, 권리 침해 신고, 보안 로그 등 법령상 또는 운영상 보관이 필요한 정보는 목적과 기간을 명확히 정해 분리 보관합니다.
- 커뮤니티 게시물은 탈퇴 후에도 서비스의 연속성과 다른 이용자의 이용 기록 보호를 위해 익명 처리 후 보존될 수 있습니다. 다만 개인정보가 포함된 게시물은 삭제 요청 절차를 제공합니다.

## 9. 보안 원칙

- 비밀번호는 인증 제공자의 안전한 해시 저장 방식을 사용하며 평문으로 저장하지 않습니다.
- 운영자, 검수자, 일반 회원의 권한을 분리합니다.
- 관리자 화면 접근과 중요 변경 이력은 로그로 남깁니다.
- 운영 데이터베이스와 개발·테스트 데이터베이스를 분리합니다.
- 개인정보 유출, 오발송, 권한 설정 오류가 발생했을 때의 대응 절차를 문서화합니다.
- 개인정보가 포함된 증빙 파일은 원본 저장을 지양하고, 불가피하게 저장하는 경우 접근 권한과 보관 기간을 엄격히 제한합니다.

## 10. 출시 전 개인정보 체크리스트

- 개인정보 처리방침 공개
- 이용약관 공개
- 필수 동의와 선택 동의 UI 분리
- 마케팅 수신 동의 기본값 미동의
- 쿠키, 분석 도구, 로그 수집 안내
- 인증, 데이터베이스, 결제 등 외부 서비스 처리위탁 여부 확인
- 국외 이전 여부 확인
- 탈퇴와 개인정보 삭제·익명 처리 동작 확인
- 관리자 권한과 접근 통제 확인
- 만 14세 미만 가입 제한 또는 법정대리인 동의 절차 확인
