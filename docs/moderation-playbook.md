# KVLE 모더레이션 운영 매뉴얼

시행일: 2026년 6월 9일

본 문서는 KVLE 커뮤니티의 신고, 임시 비공개, 운영자 삭제, 복구, 계정 제한, 이의제기 처리 기준을 정리한 내부 운영 기준입니다. 공개 약관과 커뮤니티 가이드라인의 톤을 따르며, 사용자에게는 가능한 한 판단 사유와 다음 절차를 짧고 명확하게 안내합니다.

## 1. 운영 원칙

- 학습 목적과 직접 관련된 토론은 최대한 보존합니다.
- 권리 침해, 개인정보 노출, 시험 부정행위 조장, 불법 정보, 보안 위협은 신속히 임시 비공개합니다.
- 단순 의견 차이, 풀이 방식 차이, 근거 있는 반박은 삭제 대상이 아닙니다.
- 운영 조치는 `admin_audit_logs`에 남기고, 가능한 범위에서 신고자와 작성자에게 결과를 안내합니다.
- 내부 상태값은 `visible`, `hidden_by_author`, `hidden_by_votes`, `blinded_by_report`, `removed_by_admin`을 유지하되, 사용자-facing 문구는 `표시 중`, `작성자 삭제`, `평가로 접힘`, `임시 비공개`, `운영자 삭제`를 사용합니다.

## 2. 상태 문구 기준

| 내부 상태 | 사용자 문구 | 관리자 문구 | 의미 |
|---|---|---|---|
| `visible` | 표시 중 | 표시 중 | 정상 노출 |
| `hidden_by_author` | 작성자가 삭제한 글 | 작성자 삭제 | 작성자 본인 삭제 |
| `hidden_by_votes` | 추천/비추천 평가로 접힌 글 | 평가로 접힘 | 품질 신호로 기본 접힘 |
| `blinded_by_report` | 신고 접수로 임시 비공개 중 | 임시 비공개 | 신고 누적 또는 운영자 판단으로 검토 전 비공개 |
| `removed_by_admin` | 운영자에 의해 삭제된 글 | 운영자 삭제 | 검토 후 삭제 확정 |

## 3. 신고 유형별 1차 처리

| 신고 유형 | 1차 조치 | 목표 처리 시간 | 기준 |
|---|---|---:|---|
| 개인정보 노출 | 즉시 임시 비공개 | 24시간 이내 | 이름, 학교, 학번, 연락처, 사진, 계정 정보 등 식별 가능 정보 |
| 저작권 침해 | 임시 비공개 또는 운영자 삭제 | 72시간 이내 | 교재, 강의, 유료 모의고사, 타인 해설의 무단 게시 |
| 명예훼손/권리침해 | 임시 비공개 | 24시간 이내 | 특정 개인/집단 식별 및 평판 훼손 가능성 |
| 시험 부정행위 | 즉시 임시 비공개 | 24시간 이내 | 비공개 시험 정보, 부정행위 방법, 보안 위반 조장 |
| 허위/잘못된 정보 | 표시 유지 또는 임시 비공개 | 72시간 이내 | 근거 없는 정답 단정, 반복적 오답 전파 |
| 혐오/괴롭힘 | 임시 비공개 또는 운영자 삭제 | 72시간 이내 | 욕설, 차별, 인신공격, 성희롱 |
| 광고/스팸 | 운영자 삭제 | 72시간 이내 | 반복 홍보, 외부 유도, 무관한 링크 |
| 기타 | 사안별 판단 | 72시간 이내 | 가이드라인과 학습 목적 기준으로 판단 |

## 4. 처리 결과 기준

- 유지: 가이드라인 위반이 아니거나, 반박 가능한 학습 의견인 경우.
- 임시 비공개: 권리침해, 개인정보, 명예훼손, 시험 보안 등 추가 확인이 필요한 경우.
- 운영자 삭제: 명백한 위반이 확인되었거나 복구해도 학습 목적을 해치는 경우.
- 작성자 경고: 위반 정도가 낮거나 최초 위반인 경우.
- 작성 제한/이용 정지: 반복 위반, 보복성 신고, 운영 방해, 개인정보/권리침해 반복이 있는 경우.

## 5. 안내 문구 템플릿

- 신고 접수: `신고가 접수되었습니다. 운영자가 검토할게요.`
- 임시 비공개: `신고 접수로 임시 비공개 처리되었습니다. 운영자 검토 후 복구 또는 삭제될 수 있습니다.`
- 신고 인정: `신고하신 글을 검토한 결과 가이드라인 위반으로 판단되어 운영자 삭제 처리했습니다.`
- 신고 기각: `신고하신 글을 검토한 결과 현재 기준에서는 위반으로 보기 어려워 표시 상태를 유지했습니다.`
- 작성자 안내: `회원님의 글이 임시 비공개 처리되었습니다. 조치에 이의가 있으면 통지를 받은 날부터 14일 이내에 이의 제기를 할 수 있습니다.`
- 복구 안내: `검토 결과 위반으로 보기 어려워 글을 다시 표시했습니다.`

## 6. 이의제기 처리

- 작성자는 조치 통지를 받은 날부터 14일 이내에 이의 제기를 할 수 있습니다.
- 이의제기에는 대상 글, 조치 일시, 이의 사유, 추가 근거를 포함하도록 안내합니다.
- 운영자는 기존 처리자가 아닌 운영자 또는 검수자가 재검토하는 것을 원칙으로 합니다.
- 결과는 유지, 복구, 부분 수정 요청, 삭제 확정 중 하나로 기록합니다.

## 7. 출시 전 체크리스트

- 신고 큐에서 `임시 비공개`, `운영자 삭제`, `복구`가 각각 동작한다.
- 모든 관리자 조치가 `admin_audit_logs`에 기록된다.
- 신고자와 작성자 알림 문구가 약관, 개인정보 정책, 커뮤니티 가이드라인과 같은 톤이다.
- 개인정보 포함 게시물의 삭제 요청 경로가 안내된다.
- 탈퇴 시 댓글, 신고 내역, 학습 데이터가 삭제/익명화/분리보관 기준대로 처리된다.
